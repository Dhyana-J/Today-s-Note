* 보조스트림
보조스트림은 기반스트림 생성 이후에 사용 가능하다.


// 객체 자체를 파일에 출력(저장)할 때 사용되는 스트림 클래스

// FileOutputStream : 파일 출력 시 사용되는 기반 스트림

* 파일에 기록된 객체 단위를 입력받기 위한 스트림 클래스

// FileInputStream : 파일에 기록된 데이터를 입력받기 위한 기반스트림
